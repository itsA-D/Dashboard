🔷 What is Zaggle (in simple terms)

Zaggle = a business spend management platform (FinTech SaaS)

👉 It helps companies:

Track all employee expenses
Control company spending
Automate approvals, reimbursements, payments
Give corporate cards / benefits / rewards

📌 Instead of Excel sheets + emails + manual approvals → everything is done in one dashboard
📌 It gives real-time visibility + automation + analytics

🧠 Core Idea (VERY IMPORTANT)

Zaggle replaces this messy flow:

Employee spends money → keeps receipts → submits Excel → manager approves → finance reimburses

with:

Employee uses Zaggle → expense auto-recorded → approval auto → tracked → controlled

🏗️ Main Products inside Zaggle (ecosystem)

Zaggle is not just one dashboard — it’s a suite:

Zaggle Save → employee expenses & reimbursements
Zoyer → business payments & accounts payable
ZatiX → analytics dashboard
Propel → rewards & employee engagement

👉 All connected in one system

👨‍💼 EMPLOYEE SIDE (Dashboard Flow)
1. Getting Access
Employee gets:
Login (web/app)
Or corporate prepaid/credit card
2. Spending Money

There are 2 ways:

A. Corporate Card (Most common)
Company gives Zaggle card
Employee spends directly (travel, food, fuel etc.)
No out-of-pocket expense

👉 Transactions are automatically recorded

B. Personal Spend (Reimbursement)
Employee pays from own money
Uploads:
Receipt photo (OCR auto-detects data)
3. Submitting Expense

Employee dashboard lets them:

Categorize expense (travel / food / etc.)
Add notes
Submit claim

📌 Mobile-friendly → can do on the go

4. Tracking Status

Employee can see:

Pending approval
Approved / rejected
Reimbursement status
5. Benefits & Rewards

Employee can also:

Use meal cards / vouchers
Earn reward points
Access perks (lounges, etc.)
👨‍💻 ADMIN / COMPANY SIDE (Dashboard Flow)

This is where the real power is.

1. Setting Policies

Admin defines:

Expense limits (₹ per day / category)
Allowed vendors
Travel policies
Per diem rules

👉 System enforces compliance automatically

2. Issuing Cards & Budgets

Admin can:

Issue corporate cards to employees
Set spending limits per employee/team
Allocate budgets per department/project

👉 Prevents overspending at source

3. Approval Workflow

Instead of manual emails:

Employee submits expense
System routes to:
Manager → Finance → Final approval

👉 Fully automated workflows

4. Real-Time Dashboard (MOST IMPORTANT)

Admin sees:

Total company spend
Department-wise spend
Employee-wise spend
Category trends (travel, SaaS, etc.)

👉 Full visibility across company

5. Analytics & Insights

Zaggle dashboard shows:

Who is spending the most
Where money is going
Cost-saving opportunities

👉 Helps decision-making

6. Automated Accounting & Payments
Expense data syncs with accounting systems
Reimbursements processed automatically
Vendor payments managed (Zoyer)

👉 Reduces manual finance work

7. Compliance & Audit
Every transaction logged
Timestamp + audit trail
Policy violations flagged

👉 100% transparency
